ARGELLI Ice Shell

Форма для льда в виде ракушки — свежесть и уход каждый день

О продукте

ARGELLI Ice Shell — многоразовая форма для льда из пищевого силикона. Используется для криомассажа лица, снятия отёков и тонизации кожи. Идеально для утреннего ухода.

Преимущества

  • Эстетичный дизайн и компактный размер
  • Безопасный силикон (BPA free)
  • Подходит для чувствительной кожи
  • Многоразовое использование

Как использовать

  1. Налейте в форму воду или настой
  2. Заморозьте на 3–4 часа
  3. Массируйте кожу льдом по массажным линиям
